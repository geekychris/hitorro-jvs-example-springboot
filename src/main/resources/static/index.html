Pipeline Results

Pipeline results will appear here. Choose a mode and pipeline type above, then click Run Pipeline. Modes: - Hitorro Iterator: Uses JSONIterator -> map() -> JvsEnrichMapper -> JsonSink - Java Streams: Uses Stream.map() -> collect() -> writeNdjsonFile() - Iterator -> Stream: Uses JSONIterator.toStream() bridge - Stream -> Iterator Sink: Uses AbstractIterator.fromStream() -> .sink(JsonSink) Pipelines: - Enrich: Applies NLP enrichment (NER, segmentation, stemming) to all documents - Translate: Translates title and body MLS fields via Ollama - Combined: Enriches first, then translates

Translate → Enrich → Index Pipeline

Pipeline: Load dataset → Translate MLS fields to target languages (Ollama) → Enrich with NLP (segmentation, NER, stemming) across all languages → Store in RocksDB KVStore + Index in Lucene. The index stores only projected fields for search; full documents are retrieved from the KV store.
